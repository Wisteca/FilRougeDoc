Detailed Description

This singleton manages events processing.

It puts each event in a queue on call of pushEvent(). On each call of processEvents(), all waiting events are processed one by one in the order they was added (FIFO). When processing an event, the method onEvent() of each listener present in the list is called. Use addListener() and removeListener() to register/unregister a listener.

Example :

EventManager::start(); // Don't forget to call start() to create the singleton, otherwise getInstance() will return nullptr
EventManager::getInstance()->pushEvent(new AttackEvent()); // Push an event in the queue
EventManager::getInstance()->processEvents(); // onEvent() of each listener is called with a pointer to the AttackEvent as argument
EventManager::end(); // Destroy the manager, free memory
