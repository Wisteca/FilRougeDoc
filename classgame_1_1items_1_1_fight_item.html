Detailed Description

A FightItem is an object that has a role in combat.

To use a FightItem an entity must equip it (become its owner/user).

The canEquip() method checks if an entity is able to equip the item by reviewing its skills and compare them to the requirements map. The equip() method equip the entity if canEquip() returns true.

Once the entity is equipped, the FightItem will apply some modificators to its skills.
