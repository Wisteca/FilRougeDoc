enum game::skills::SkillLevel : int
strong

This enum specifies the difficulty to improve a skill.

The value of each enum type is direclty the cost in experience points to increase a skill. SkillsSet has a map<SkillType, SkillLevel> that specifies the difficulty to improve each skill.
