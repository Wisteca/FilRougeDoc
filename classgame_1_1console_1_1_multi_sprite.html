Detailed Description

MultiSprite implementation. This Drawable object can display different Sprite regarding the index value.

This can be used to display an entity that moves right, left, up and down for example. Just give the names of all Sprite to load and choose which one will be displayed with setCurrentSpriteIndex().

this object is just a list of sprites that takes care of calling the draw() function of the current Sprite object according to the selection index

Note that all sprites must have the same size.
