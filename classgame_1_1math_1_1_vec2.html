Detailed Description

template<typename T = float>
class game::math::Vec2< T >

2D vector (x,y) with useful methods to use it

This is a template class, it is possible to create vector of differents types. Default type is float.

There is a cast operator implemented, that means :

Vec2<int> v1(5, 5); // Vector of int type
Vec2<> v2(5.1, 5.1); // Vector of float type
cout << static_cast<Vec2<int>>(v2) << endl; // will print Vec2 {x = 5, y = 5}
cout << boolalpha << (v2 == v1) << endl; // Is possible (implicit cast) and will print false because v1 will be casted in type of v2
cout << boolalpha << (v1 == v2) << endl; // Be careful, this will print true because v2 will be cated to int,int -> 5,5 == 5,5
v2 -= Vec2<>(.1, .1);
cout << boolalpha << (v2 == v1) << endl; // Will print true
