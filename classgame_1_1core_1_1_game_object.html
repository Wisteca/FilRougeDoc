Detailed Description

An simple object, everything that has a role in the game must be an instance of this or inherit from this.

A GameObject has a position in the world (in WC = world coordinates), a size (in WC or in SC for the Sprite), a radius and a MultiSprite.

TODO : The radius is used to avoid two object be one on the other. If the distance between two object is smaller than entity1.radius + entity2.radius, the move is cancelled

The MultiSprite is the appearance of the object.
