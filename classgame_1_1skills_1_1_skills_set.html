Detailed Description

A set of all skills each Entity has with useful methods to manage them.

This is basically just a list of Skill.
This class has two main methods :

  • SkillsSet::improveSkill() to improve a skill with experience points
  • SkillsSet::computeActualValue() to obtain the calculated final value of a skill present in the set

bool game::skills::SkillsSet::deserialize ( rapidxml::xml_node<> *  node)
overridevirtual

Deserialize a SkillsSet from a xml node.

The way a SkillsSet is saved in a node is pretty simple. There is a skill node for each SkillType that has the name and the base value of the skill as attributes. The temp value of a skill and SkillLevel map aren't saved. It looks like this : <SkillsSet> <skill name="skill1" value="10"> <skill name="skill2" value="10"> <skill name="skill3" value="10"> ... </SkillsSet>

Parameters
nodeXML node to deserialize
Returns
false if everything happens fine, true otherwise

Implements game::io::IOSerializer.

Improve a skill with exp points.

Parameters
typeSkillType of the skill you want to improve
quantityAmount of points to increase/decrease (use negative value to decrease)
expA reference on the exp points variable that will be used to determine if it's possible to improve desired skill
Returns
true if improvement succeeded

This is a little example how it works :

int exp = 6; // You have 6 exp points
// You want to increase your STAMINA skill by 3 points
bool success = set.improveSkill(SkillType::STAMINA, 3, exp);
// Case 1 : STAMINA is specified as EASY to improve :
// -> success = true, exp = 3 (cost = 3 points)
// Case 2 : STAMINA is specified as NORMAL to improve :
// -> success = true, exp = 0 (cost = 6 points)
// Case 3 : STAMINA is specified as HARD to improve :
// -> success = false, exp = 6 (cost = 9 points, you don't have enough)
// Assuming case 2, you now have 0 exp point and your STAMINA increased by 3 points
// You now want to get your exp back to spend it somewhere else
set.improveSkill(SkillType::STAMINA, -3, exp);
// Now you lost 3 STAMINA points and your exp is 6 again.
