Detailed Description

An Entity is the base class of every "living thing", like a player or a monster.

An Entity has a SkillsSet that contains all entity specificity. It is used when fighting, moving, etc

An Entity has health points, when they fall below 0, the entity die. The Suto points are used for launching spells.
