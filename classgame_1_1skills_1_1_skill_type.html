Detailed Description

Skills enumeration.

This is a Java-like enum, each skill has attributes and methods. The reason of this architecture is that each skill have a complete name and depends of other skills in percent. So it is possible to access theses members like this :

std::string name = SkillType::STAMINA.getName();
int dependencie = SkillType::WEAPON_HANDLING.getDependencie(SkillType::STAMINA);
