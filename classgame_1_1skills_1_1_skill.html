Detailed Description

This class contain a skill type, a base value for this skill and a temporary value used by buffs/debuffs.

Objects of this class are created and stored in an array managed by SkillsSet. This class has three attributes :

  • The type of the skill, specified by an enum SkillType
  • The base value of the skill
  • A temporary value added to the base value, used for temporary buff/debuff -> For example, a spell can boost a specific skill for one minute
