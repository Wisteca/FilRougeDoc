Detailed Description

A Sprite is an ASCII image that can be displayed by the ConsoleEngine.

A Sprite has a position in the screen that can be accessed with setPosition() and getPosition() methods.

The Sprite is loaded from a file that has to respect this syntax :

29
8
s
_____ _ _
/ ____| (_) |
| (___ _ __ _ __ _| |_ ___
\___ \| '_ \| '__| | __/ _ \
____) | |_) | | | | || __/
|_____/| .__/|_| |_|\__\___|
| |
|_|

Where 29 is the width and 8 the height (number of character). The character s will be used to display spaces. By default all spaces ' ' in the file are blank (or alpha = 0), so when two Sprite are drawn one over the other there will be transparency. If this behavior isn't desired, the "space char" can be used to tell "here is a space and not a blank".

It is possible to create a Sprite without loading from a file. You can create a Sprite by specifying the size and the space char, then use setPixel() method to draw on it.

It is possible to draw text on a Sprite by using writeText() method.
