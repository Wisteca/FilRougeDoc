Detailed Description

This singleton manages all the game.

Non-exhaustive list of managements :

  • ConsoleEngine
  • Contains the list of objects controllers
  • Reference on the World object
  • Camera position and owner entity (the entity the camera follow)
  • List of loaded sprites
  • HUD for displaying informations
  • TODO : play/pause
  • Level generation (place objects in the world)

This is the core class of the game.
